Velkommen til dokumentasjonen for **Digdir**-profilen. Denne siden er skrevet i
MDX og rendres gjennom det samme systemet som designsystemet.no.

## Installasjon

Installer pakkene du trenger:

```bash
pnpm add @digdir/designsystemet-react @digdir/designsystemet-css
```

## Bruk en komponent

Importer og bruk komponenter slik:

```tsx
import { Button } from '@digdir/designsystemet-react';

export function Example() {
  return <Button>Trykk her</Button>;
}
```
